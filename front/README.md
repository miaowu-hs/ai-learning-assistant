# Frontend

AI 学习助手前端，基于 Vue 3 + Vite + TypeScript + Element Plus 构建。

## 技术栈

- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Axios
- Element Plus

## 启动方式

```bash
npm install
npm run dev
```

默认开发地址：`http://localhost:5173`

## 说明

- 开发环境已通过 Vite 代理将 `/api` 请求转发到本地后端 `http://localhost:8080`
- 页面默认中文
- 当前只实现后端已开放接口对应的功能
